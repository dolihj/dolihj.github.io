# file link in obsidian
```diff
[[ file ]]
[[ file#section]]
[[ file / header]]
[[ file^linehash]]
![[ embeded_file ]]
example: 
  ![[What_is_IPSec#ref]]

   [[What_is_IPSec#^a80e6f]]
   ![[What_is_IPSec#^a80e6f]] 

filepath (outside of vault)
 format :  [link_text](file:///Users/file_path )
 example: 
  [orientation](file:///Users/parkhj1/Documents/0425orientation.md)
```
